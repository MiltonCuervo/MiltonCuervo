## Sobre mí

Soy estudiante de Ingeniería de Sistemas enfocado en el desarrollo de soluciones que integren backend moderno, análisis de datos y automatización empresarial.

Diseño e implemento APIs REST, estructuras de datos escalables y procesos automatizados que transforman tareas manuales en sistemas estructurados.

Mi enfoque combina:

- Desarrollo backend (Spring Boot, FastAPI)
- Análisis y transformación de datos (Python, Pandas, NumPy)
- Implementación de soluciones con Microsoft Power Platform
- Integración de servicios y APIs

Busco contribuir en entornos donde el software no solo funcione, sino que genere impacto operativo real.
